READ ME




This package version includes the program for the following requirement mentioned for Project-3:

Extend project 2 and add the following features:

1. Negative numbers
2. Division operation: quotient (/) and remainder (%)
3. Square root (just the integer part)




NOTE: 
* The input is taken from an input file- “project_3_inputFile1.txt”, and the output will be displayed on the console.
* Invalid expressions will give the output as “Syntax Error”.
* Invalid operators will give the output as “Unknown Operator”.
* BigNumberOperation.java acts as the main class to perform the functionality.
* By default the class uses the base as 100000000. The base can be changed any value from [10^1, 10^2, 10^3,........10^9].
